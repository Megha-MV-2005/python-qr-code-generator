## Overview
This project is a Python-based QR Code Generator developed to create scannable QR codes using external libraries.  
The generated QR code encodes a Google Drive link that points to my Python completion certificate, making it easy to share and access digital credentials.

## Purpose
The main purpose of this project is to demonstrate my understanding of:
- Python programming fundamentals
- Working with third-party Python libraries
- Converting real-world data into usable digital outputs

This project also reflects how Python can be used to build simple yet practical applications.

## Features
- Generates QR codes from a given text or URL
- Saves the generated QR code as a PNG image
- Uses external Python libraries for QR generation
- Produces a scannable and shareable output

## Technologies Used
- Python
- qrcode library
- Pillow (PIL)

## How It Works
The application takes a Google Drive link as input data and encodes it into a QR code.  
Once executed, the program generates a PNG image file containing the QR code, which can be scanned using any QR scanner to open the linked resource.
